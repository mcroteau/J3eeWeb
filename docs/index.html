Qio

Qio is a web framework built for both the Java EE and Jakarta EE specifications.

Add:

            
<dependency>
    <groupId>dev.j3ee</groupId>
    <artifactId>qio</artifactId>
    <version>1.4</version>
</dependency>

Then configure via : src/main/resources/qio.props settings file.

            
qio.dev=true
qio.assets=assets,media
qio.properties=qio.props

... and you're ready to go.
