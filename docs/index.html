J3ee

Modern Java web development. It is a spin on the old J2ee acronym that SAP & Microsoft are quietly marginalizing! J2ee is beautiful.

A8i

An Open Source Java Web Server built on the com.sun.net.httpserver.HttpExchange. An alternative to Tomcat & Jetty with built in dependency injection, request routing and data management. Also, if you are tired of deploying war files, A8i projects run as exploded jars or as a single fat uber jar!

Chico

Chico is a security plugin built for Jakarta EE based projects. It is free to download and the latest version can be found on either github or mavenrepository.com

            
<dependency>
    <groupId>dev.j3ee</groupId>
    <artifactId>chico</artifactId>
    <version>0.1</version>
</dependency>

Chico is based off of Parakeet. Parakeet is 9 versions deep so it is a pretty stable security plugin. Don't let the version number throw you off.

Papi

The latest version can be found here : https://github.com/mcroteau/Papi. Papi is a one class pooled datasource. It is super simple and easy to implement. A million times faster than a basic datasource, but not as fast as the other guys... "come on Papi, you can do it...".
